# Inspire Music

Inspire Music is a local Android music player with Liquid Glass-inspired visuals, playlist management, lyrics, and optional AI playlist generation.

## Download

Download the latest APK from the [Releases](../../releases) page.

## Features

- Local music library
- Albums, artists, songs, playlists, and favorites
- Liquid Glass-inspired bottom bar, mini player, menus, and now playing view
- Lyrics support with local and optional online lyric search
- Playlist backup, import, and device-to-device transfer through Android sharing
- Optional AI playlist generation with user-provided API settings

## AI Notice

AI features are optional. This app does not include any built-in API key or hosted AI service.

To use AI features, configure your own provider in the app settings:

- API base URL
- API key
- Model name

You are responsible for complying with the terms of your chosen AI provider and local laws.

## Installation

1. Download the APK from Releases.
2. Open it on your Android device.
3. Allow installation from this source if Android asks.
4. Grant music/media permission after launching the app.

## Privacy

The core music player works locally on your device. Optional online features may send requests to the service provider you configure.
